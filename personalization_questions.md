# AIフィルタリングのためのパーソナライズ質問

AIによるニュースフィルタリングの精度を高め、あなたにとって真に価値ある情報を提供するために、以下の質問にご協力ください。これらの情報は、あなたの「情報ペルソナ」を構築し、フィルターロジックを最適化するために使用されます。

## 1. 基本情報と役割
1.  現在の主な職業／役割は何ですか？ (例: ソフトウェア開発マネージャー、データサイエンティスト、スタートアップ創業者)
2.  主要な業務分野または興味のある業界は何ですか？ (例: 自動車業界の組み込みシステム、FinTechのブロックチェーン技術、Webマーケティング)
3.  現在、取り組んでいるプロジェクトや研究テーマがあれば教えてください。それがニュース収集にどう影響しますか？

## 2. 情報収集の課題と好み
1.  現状のニュース収集で、特に「疲弊する」「無駄だと感じる」情報はどのようなものですか？具体的なキーワードやトピックを挙げてください。(例: ダイエット記事、芸能ゴシップ、古い技術トレンドの再放送、投資勧誘)
2.  「これだけは絶対に見逃したくない」という最重要トピックは何ですか？ (例: 競合他社の最新技術動向、特定のプログラミング言語のメジャーアップデート、法改正情報)
3.  ニュース記事を読む際、どのような情報に最も価値を感じますか？ (例: 結論、背景、具体的な実装例、市場への影響、自分の仕事への示唆)
4.  特定の情報源（Webサイト、ブログ、雑誌など）で、常にチェックしているものはありますか？ (RSSの有無も分かれば教えてください)

## 3. 配信と利用シーンの最適化
1.  ニュースの要約は、どのくらいの長さが理想的ですか？ (例: 1行、3行、短文の箇条書き)
2.  「電車での視覚確認」と「歩行中の耳学」のどちらをより重視しますか？または、それぞれのシーンでどのような情報を求めていますか？
3.  「自分だけでなく、チームや同僚にも共有すべき情報」の重要度をAIに判定させることに興味がありますか？もしそうであれば、どのような基準で共有すべきと判断してほしいですか？
4.  このサービスを通じて、最終的にどのような状態を実現したいですか？ (例: 情報過多によるストレスからの解放、常に業界の最前線にいる感覚、新しいアイデアの創出)

これらの質問への回答は、AIフィルターがあなたの「専属秘書」のように機能するための重要な手がかりとなります。ご協力ありがとうございます。
